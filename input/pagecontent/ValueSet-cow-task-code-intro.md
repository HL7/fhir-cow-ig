This value set can be used in any Task's `Task.code`, but the codes defined by this IG are typically intended for use in dedicated Tasks that support specific workflow steps — such as bidding for fulfilment of a request — rather than in the general-purpose Coordination Task.

The value set combines the standard FHIR [task-code](https://hl7.org/fhir/codesystem-task-code.html) codes with additional codes defined by this IG to address workflow patterns not yet covered by the base specification.
